# 📊 Superstore Sales Dashboard

## Overview
An interactive Power BI dashboard built using the Superstore dataset to analyze sales performance, profit, quantity, and business growth across categories, products, regions, and time.

## Objectives
- Monitor overall sales performance.
- Analyze sales trends over time.
- Identify top-performing products and categories.
- Visualize regional sales distribution.

## Tools Used
- Microsoft Power BI
- Power Query
- DAX

## Key KPIs
- **Total Sales:** 2.30M
- **Total Profit:** 286.40K
- **Growth Rate:** 4.10%
- **Total Quantity:** 38K

## 💡 Key Insights

- Technology category generated the highest sales among all product categories.
- Sales showed a consistent upward trend across the quarters.
- A few top-performing products contributed significantly to total revenue.
- Regional analysis highlighted differences in sales performance across locations.
- Interactive filters enable quick analysis of category-wise sales performance.
